# תכנון עלויות ותקציב - אתר חנות תכשיטים

## 📊 עלויות אחסון (Hosting)

### אפשרויות אחסון מומלצות:

1. **Vercel (מומלץ לפרויקט זה)**
   - תכנית חינמית: 0 ₪/חודש
   - Pro: ~$80 (~280 ₪)/חודש
   - יתרונות: אופטימיזציה אוטומטית, CI/CD, תמיכה ב-React
   
2. **Netlify**
   - תכנית חינמית: 0 ₪/חודש
   - Pro: $19 (~67 ₪)/חודש
   - יתרונות: פריסה מהירה, SSL חינמי
   
3. **AWS (Amazon Web Services)**
   - EC2: החל מ-$3.5 (~12 ₪)/חודש
   - S3 + CloudFront: ~$5-10 (~18-35 ₪)/חודש
   - יתרונות: גמישות רבה, סקלביליות
   
4. **Google Cloud Platform**
   - App Engine: החל מ-$5 (~18 ₪)/חודש
   - Firebase Hosting: תכנית חינמית זמינה
   
5. **Digital Ocean**
   - Droplet בסיסי: $6 (~21 ₪)/חודש
   - App Platform: $5 (~18 ₪)/חודש

**המלצה לתחילת הדרך:** Vercel או Netlify (חינמי) ⭐

## 💳 עלויות סליקה באינטרנט

### שירותי סליקה בישראל:

1. **PayPal**
   - עמלת עסקה: 3.4% + 1.20 ₪
   - ללא דמי מנוי חודשיים
   - תמיכה בינלאומית מעולה
   
2. **Stripe**
   - עמלת עסקה: 2.9% + 1.20 ₪
   - ללא דמי מנוי
   - API מתקדם ופשוט לשילוב
   
3. **Tranzila (טרנזילה)**
   - דמי מנוי: ~150 ₪/חודש
   - עמלת עסקה: 2.5-3%
   - שירות ישראלי מקומי
   
4. **Cardcom (קארדקום)**
   - דמי מנוי: ~100-200 ₪/חודש
   - עמלת עסקה: 2-3%
   - תמיכה בעברית
   
5. **Meshulam (משולם)**
   - דמי הקמה: ~500 ₪
   - דמי מנוי: ~100 ₪/חודש
   - עמלת עסקה: 2.5%

**המלצה:** Stripe (גמיש ונוח) או Tranzila (ישראלי ומקומי) ⭐

### חישוב דוגמה לעלויות סליקה:
- מכירה של 1,000 ₪
- עם Stripe (2.9% + 1.20 ₪) = 30.20 ₪ עמלה
- נשאר לך: 969.80 ₪
